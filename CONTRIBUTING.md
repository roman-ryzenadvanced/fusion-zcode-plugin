# Contributing to Fusion

Thank you for your interest in contributing to Fusion! This document provides guidelines and instructions for contributing.

## Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## How to Contribute

### Reporting Bugs
1. Check if the bug has already been reported in Issues
2. Create a new issue with clear description and reproduction steps

### Suggesting Features
Open an issue describing the feature and use case.

### Pull Requests
1. Fork the repository
2. Create your feature branch: `git checkout -b feature/amazing-feature`
3. Make your changes
4. Run tests: `npm run typecheck`
5. Commit and push
6. Open a Pull Request

## Development Setup
```bash
npm install
npm run build
npm run typecheck
```

No secrets/credentials in code - use environment variables.
